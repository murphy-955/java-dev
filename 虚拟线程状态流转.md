stateDiagram-v2
[*] --> NEW

    %% ========== 生命周期 ==========
    NEW --> STARTED : Thread.start()
    STARTED --> RUNNING : 首次调度\n(mount/thaw)
    STARTED --> TERMINATED : 启动失败
    RUNNING --> TERMINATED : 执行完毕

    %% ========== Park (无限期) ==========
    RUNNING --> PARKING : LockSupport.park()
    PARKING --> PARKED : cont.yield 成功\n(freeze/unmount)
    PARKING --> PINNED : cont.yield 失败\n(持有 monitor/JNI)
    PARKED --> UNPARKED : unpark()
    PINNED --> RUNNING : unpark()\n(同载体继续，未 unmount)
    UNPARKED --> RUNNING : 重新调度\n(mount/thaw)

    %% ========== Park (定时) ==========
    RUNNING --> TIMED_PARKING : LockSupport.parkNanos()
    TIMED_PARKING --> TIMED_PARKED : cont.yield 成功\n(freeze/unmount)
    TIMED_PARKING --> TIMED_PINNED : cont.yield 失败\n(持有 monitor/JNI)
    TIMED_PARKED --> UNPARKED : unpark() / 超时
    TIMED_PINNED --> RUNNING : unpark() / 超时\n(同载体继续，未 unmount)

    %% ========== Monitor 进入 ==========
    RUNNING --> BLOCKING : synchronized\n(尝试获取 monitor)
    BLOCKING --> BLOCKED : 获取失败\n(进入 monitor 等待队列)
    BLOCKED --> UNBLOCKED : 获取成功\n(被 owner 释放后抢到)
    UNBLOCKED --> RUNNING : 重新调度\n(mount/thaw)

    %% ========== Object.wait (无限期) ==========
    RUNNING --> WAITING : Object.wait()\n(过渡态)
    WAITING --> WAIT : 进入等待队列\n(释放 monitor)
    WAIT --> BLOCKED : notify()\n(需重新抢锁)
    WAIT --> UNBLOCKED : 超时 / 中断

    %% ========== Object.wait (定时) ==========
    RUNNING --> TIMED_WAITING : Object.wait(timeout)\n(过渡态)
    TIMED_WAITING --> TIMED_WAIT : 进入等待队列\n(释放 monitor)
    TIMED_WAIT --> BLOCKED : notify()\n(需重新抢锁)
    TIMED_WAIT --> UNBLOCKED : 超时 / 中断

    %% ========== Thread.yield ==========
    RUNNING --> YIELDING : Thread.yield()
    YIELDING --> YIELDED : cont.yield 成功\n(freeze/unmount)
    YIELDING --> RUNNING : cont.yield 失败\n(原地继续，未 unmount)
    YIELDED --> RUNNING : 重新调度\n(mount/thaw)

    %% ========== 样式 ==========
    classDef lifecycle fill:#e1f5fe
    classDef park fill:#fff3e0
    classDef monitor fill:#f3e5f5
    classDef wait fill:#e8f5e9
    classDef yield fill:#ffebee

    class NEW,STARTED,RUNNING,TERMINATED lifecycle
    class PARKING,PARKED,PINNED,UNPARKED,TIMED_PARKING,TIMED_PARKED,TIMED_PINNED park
    class BLOCKING,BLOCKED,UNBLOCKED monitor
    class WAITING,WAIT,TIMED_WAITING,TIMED_WAIT wait
    class YIELDING,YIELDED yield```
