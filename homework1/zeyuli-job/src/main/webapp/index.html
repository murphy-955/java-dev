🎓 学生管理系统

📋 API 接口说明

  • GET /student/{id} - 根据ID查询学生信息
  • GET /student - 查询所有学生信息
  • PUT /student/{id} - 更新学生信息
  • POST /student - 添加新学生
  • DELETE /student/{id} - 删除学生

🔐 权限验证

除Swagger文档外，所有API请求都需要在Header中携带Authorization字段。
测试Token: valid-token-123456
